# DDD-dev-skills
Skills para agentes de IA  para desenvolvimento de acordo com padrões arquitetônicos DDD e SDD


# INSTALAÇÃO
Execute o comando abaixo na raiz do seu projeto para instalar esse repositorio como um submodulo
```bash
# Adicionar como submódulo dentro de .agents
git submodule add https://github.com/jalvest/DDD-dev-skills.git .agents/

# Atualizar submódulo
git submodule update --init --recursive
```
